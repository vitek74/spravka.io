Атрибуты

Вставляемый контент

audio :

autoplay: Указывает, что воспроизведение аудиофайла должно начинаться автоматически, как только файл будет полностью загружен.
controls: Отображает элементы управления воспроизведением, включающим в себя: запуск, паузу, полосу прокрутки и громкость звука. По умолчанию, при отсутствии атрибута, панель управления аудиофайлом отображена не будет.

embed

height: Определяет высоту вложенного содержимого в пикселях (пример: height="100").
src: Указывает адрес внешнего файла, который будет вставлен в элемент.
type: Указывает MIME-тип (спецификация форматирования сообщений и кодирования информации для передачи по интернету) содержимого элемента.
width: Определяет ширину вложенного содержимого в пикселях (пример: width="200").

video

autoplay: Указывает, что воспроизведение видеофайла должно начинаться автоматически, как только файл будет полностью загружен.
controls: Отображает элементы управления воспроизведением, включающие в себя: запуск, паузу, полосу прокрутки, громкость звука, включение полноэкранного режима, титры/субтитры (при наличии) и звуковую дорожку (при наличии).
height: Устанавливает высоту видеоплеера в пикселях (пример: height="200").
loop: Указывает, что воспроизведение будет начинаться сначала, каждый раз, после завершения (повтор воспроизведения).
muted: Позволяет отключить звук, чтобы по умолчанию видео воспроизводилось беззвучно, при желании пользователь сможет включить звук самостоятельно.
poster: Указывает URL-адрес изображения (картинки), которое будет отображаться пока видеофайл загружается или пока пользователь не нажмет кнопку воспроизведения.
Вставляемый контент

audio :

loop: Указывает, что воспроизведение будет начинаться сначала, каждый раз, после завершения (повтор воспроизведения).

object

data: Определяет путь к файлу, который будет вставлен на веб-страницу.
form: Связывает элемент object с формой.
height: Указывает высоту вставляемого контента в пикселях (пример: height="100").
name: Определяет имя для элемента, оно может использоваться для доступа к элементу в JavaScript.
type: Указывает MIME-тип (спецификация форматирования сообщений и кодирования информации для передачи по интернету) вставляемого контента.
usemap: Определяет вставляемый контент в качестве карты-изображения. Значение (обязательно должно начинаться с символа "#"), указанное в данном атрибуте, ассоциируется со значением атрибута name элемента map и создает связь между элементами object и map.
width: Указывает ширину вставляемого контента в пикселях (пример: width="100").
Вставляемый контент

audio :

preload: Указывает, какие действия должен выполнять браузер, если для проигрывателя не указан атрибут autoplay.
src: Определяет путь к нужному аудиофайлу

iframe

height: Указывает высоту элемента в пикселях.
seamless: Указывает, что содержимое iframe должно выглядеть так, словно оно является частью документа (без рамки и полос прокрутки).
src: Указывает адрес документа, который будет отображен во встроенном фрейме.
width: Указывает ширину элемента в пикселях.
name: Определяет имя для элемента.
sandbox: Применяет набор ограничений на содержимое встроенной области в качестве меры безопасности в случае, если в ней отображается ненадежный веб-сайт.

source

media: Указывает под какие типы устройств оптимизирован файл. Браузеры могут использовать этот атрибут для определения, возможно ли воспроизведение файла или нет.
src: Указывает URL-адрес медиа файла, который будет воспроизводится.
type: Определяет MIME-тип медиа ресурса.
